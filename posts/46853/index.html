张耒

诗词
Chinese Poems
Publish Date:   2022-04-11
Word Count:   2.6k
偶题·相逢记得画
宋代 · 张耒

诗文(Ancient Chinese)

相逢记得画桥头，花似精神柳似柔。
莫谓无情即无语，春风传意水传愁。

译文(Modern Chinese)

注解(Notes)

名句(Notes)

国亡身殒今何有，只留离骚在世间。——张耒《和端午》
梧桐真不甘衰谢，数叶迎风尚有声。——张耒《夜坐·庭户无人秋月明》
别离滋味浓于酒。著人瘦。此情不及墙东柳。春色年年如旧。——张耒《秋蕊香·帘幕疏疏风透》
业无高卑志当坚，男儿有求安得闲。——张耒《示秬秸》
庭户无人秋月明，夜霜欲落气先清。——张耒《夜坐·庭户无人秋月明》
长夏村墟风日清，檐牙燕雀已生成。——张耒《夏日三首·其一》
日暮北风吹雨去，数峰清瘦出云来。——张耒《初见嵩山》
落落疏帘邀月影，嘈嘈虚枕纳溪声。——张耒《夏日三首·其一》
竞渡深悲千载冤，忠魂一去讵能还。——张耒《和端午》
年来鞍马困尘埃，赖有青山豁我怀。——张耒《初见嵩山》
旅饭二年无此味，故园千里几时还。——张耒《春日》
金戈铁马从西来，郭公凛凛英雄才。——张耒《读中兴碑 / 读中兴颂碑》
情到不堪言处，分付东流。——张耒《风流子·木叶亭皋下》
城头月落霜如雪，楼头五更声欲绝。——张耒《示秬秸》
潼关战骨高于山，万里君王蜀中老。——张耒《读中兴碑 / 读中兴颂碑》
如丝苣甲饤春盘，韭叶金黄雪未干。——张耒《春日·如丝苣甲饤春盘》
木叶亭皋下，重阳近，又是捣衣秋。——张耒《风流子·木叶亭皋下》
芳草有情，夕阳无语，雁横南浦，人倚西楼。——张耒《风流子·木叶亭皋下》
久斑两鬓如霜雪，直欲渔樵过此生。——张耒《夏日三首·其一》
玉环妖血无人扫，渔阳马厌长安草。——张耒《读中兴碑 / 读中兴颂碑》
奈愁入庾肠，老侵潘鬓，谩簪黄菊，花也应羞。——张耒《风流子·木叶亭皋下》
元功高名谁与纪，风雅不继骚人死。——张耒《读中兴碑 / 读中兴颂碑》
但令一岁一相逢，七月七日河边渡。——张耒《七夕歌》
天工作民良久艰，谁知不如牛马福。——张耒《劳歌》
春色年年如旧。——张耒《秋蕊香·帘幕疏疏风透》
犹胜嫦娥不嫁人，夜夜孤眠广寒殿。——张耒《七夕歌》
惟其平生之志有所郁于中，故其余章末句，时有感激而不泄者。——张耒《上曾子固龙图书》
韩愈之文如先王之衣冠，郊庙之江鼎俎，至其放逸超卓，不可收揽，则极言语之怀巧，有不足以过之者。——张耒《上曾子固龙图书》
人间一叶梧桐飘，蓐收行秋回斗杓。——张耒《七夕歌》
异方时节三巵酒，残岁风烟一惨颜。——张耒《春日》
一线香飘金兽。——张耒《秋蕊香·帘幕疏疏风透》
别离滋味浓于酒。——张耒《秋蕊香·帘幕疏疏风透》
曾晳风乎舞雩咏而归，则游观之际未尝不在学也。——张耒《进学斋记》
曾子病而易大夫箦，则疾病之际未尝不在学也。——张耒《进学斋记》
玉容知安否？——张耒《风流子·木叶亭皋下》
向风前懊恼，芳心一点，寸眉两叶，禁甚闲愁？——张耒《风流子·木叶亭皋下》
北风吹衣射我饼，不忧衣单忧饼冷。——张耒《示秬秸》
蝶衣晒粉花枝舞，蛛网添丝屋角晴。——张耒《夏日三首·其一》
空恨碧云离合，青鸟沉浮。——张耒《风流子·木叶亭皋下》
朱阑倚遍黄昏后。——张耒《秋蕊香·帘幕疏疏风透》
楚天晚，白苹烟尽处，红蓼水边头。——张耒《风流子·木叶亭皋下》
香笺共锦字，两处悠悠。——张耒《风流子·木叶亭皋下》
百年兴废增感慨，当时数子今安在？——张耒《读中兴碑 / 读中兴颂碑》
君不见，荒凉浯水弃不收，时有游人打碑卖。——张耒《读中兴碑 / 读中兴颂碑》
帘幕疏疏风透。——张耒《秋蕊香·帘幕疏疏风透》
水部胸中星斗文，太师笔下龙蛇字。——张耒《读中兴碑 / 读中兴颂碑》
天遣二子传将来，高山十丈磨苍崖。——张耒《读中兴碑 / 读中兴颂碑》
谁持此碑入我室？使我一见昏眸开。——张耒《读中兴碑 / 读中兴颂碑》
捧盘出户歌一声，市楼东西人未行。——张耒《示秬秸》
举旗为风偃为雨，洒扫九庙无尘埃。——张耒《读中兴碑 / 读中兴颂碑》
今之所谓学者，既剽盗其皮肤，攘掇其土苴，比于古之人大可愧矣。——张耒《进学斋记》
冠而仕则冠而弃之，壮而仕则壮而弃之。——张耒《进学斋记》
故后世之君子大抵从仕数年，则言语笑貌嗜欲玩习之际，比之进取之初以儒自名者，固已大异矣。——张耒《进学斋记》
元丰之乙丑，余官于咸平，治其所居之西，即其旧而完之。——张耒《进学斋记》
既洁□新矣，于是取《诗》、《书》、古史陈于其中，暑则启扉，寒则塞向，朝夕处乎其中。——张耒《进学斋记》
余惰者也，故取古之道而名之曰进学，而书其说，庶朝夕得以自警焉。——张耒《进学斋记》
惟其言不浮乎其心，故因其言而求之，则潜德道志，不可隐伏。——张耒《上曾子固龙图书》
盖古之人不知言则无以知人，而世之惑者，徒知夫言与德二者不可以相通，或信其言而疑其行。——张耒《上曾子固龙图书》
是徒知其一，而不知夫君子之文章，固出于其德，与夫无其德而有其言者异位也。——张耒《上曾子固龙图书》
某之初为文，最喜读左氏、《离骚》之书。——张耒《上曾子固龙图书》
丘明之文美矣，然其行事不见于后，不可得而考。——张耒《上曾子固龙图书》
故其文如明珠美玉，丽而可悦也；如秋风夜露，凄忽而感恻也；如神仙烟云，高远而不可挹也。——张耒《上曾子固龙图书》
惟其言以考其事，其有不合者乎？——张耒《上曾子固龙图书》
自三代以来，最喜读太史公、韩退之之文。——张耒《上曾子固龙图书》
司马迁奇迈慷慨，自其少时，周游天下，交结豪杰。——张耒《上曾子固龙图书》
其学长于讨论寻绎前世之迹，负气敢言，以蹈于祸。——张耒《上曾子固龙图书》
故其文章疏荡明白，简朴而驰骋。——张耒《上曾子固龙图书》
此情不及墙东柳。——张耒《秋蕊香·帘幕疏疏风透》
廊上月华如昼。——张耒《秋蕊香·帘幕疏疏风透》
曾奉龙旗典邦礼，岁穷祠祀少休闲。——张耒《春日》
谁持此碑入我室？——张耒《读中兴碑 / 读中兴颂碑》
使我一见昏眸开。——张耒《读中兴碑 / 读中兴颂碑》
神宫召集役灵鹊，直渡天河云作桥。——张耒《七夕歌》
桥东美人天帝子，机杼年年劳玉指。——张耒《七夕歌》
织成云雾紫绡衣，辛苦无欢容不理。——张耒《七夕歌》
帝怜独居无与娱，河西嫁得牵牛夫。——张耒《七夕歌》
自从嫁后废织纴，绿鬓云鬟朝暮梳。——张耒《七夕歌》
贪欢不归天帝怒，谪归却踏来时路。——张耒《七夕歌》
别多会少知奈何，却忆从前恩爱多。——张耒《七夕歌》
匆匆离恨说不尽，烛龙已驾随羲和。——张耒《七夕歌》
河边灵官晓催发，令严不管轻离别。——张耒《七夕歌》
空将泪作雨滂沱，泪痕有尽愁无歇。——张耒《七夕歌》
寄言织女若休叹，天地无情会相见。——张耒《七夕歌》
退之之于唐，盖不试遇矣。——张耒《上曾子固龙图书》
自唐以来，更五代之纷纭。——张耒《上曾子固龙图书》
宋兴，锄叛而讨亡。——张耒《上曾子固龙图书》
及仁宗之朝，天下大定，兵戈不试，休养生息，日趋于富盛之域。——张耒《上曾子固龙图书》
士大夫之游于其时者，谈笑佚乐，无复向者幽忧不平之气，天下之文章稍稍兴起。——张耒《上曾子固龙图书》
而后四方学者，始耻其旧而惟古之求。——张耒《上曾子固龙图书》
而欧阳公于是时，实持其权以开引天下豪杰，而世之号能文章者，其出欧阳之门者居十九焉。——张耒《上曾子固龙图书》
而执事实为之冠，其文章论议与之上下。——张耒《上曾子固龙图书》
闻之先达，以谓公之文其兴虽后于欧公，屹然欧公之所畏，忘其后来而论及者也。——张耒《上曾子固龙图书》
某自初读书即知读执事之文既思而思之广求远访以日揽其变呜呼如公者真极天下之文者欤！——张耒《上曾子固龙图书》
独爱诗篇超物象，只应山水与精神。——张耒《岁暮福昌怀古四首其三{李贺宅}》
梦觉隔窗残月尽，五更春鸟满山啼。——张耒《福昌官舍》
爱尔千岁姿，藏身一莲叶。——张耒《暇日六咏其五{龟}》
暑天三月元无雨，云头不合惟飞土。——张耒《劳歌》
深堂无人午睡余，欲动儿先汗如雨。——张耒《劳歌》
忽怜长街负重民，筋骸长彀十石弩。——张耒《劳歌》
半衲遮背是生涯，以力受金饱儿女。——张耒《劳歌》

作者(Author)

张耒（1054—1114年），字文潜，号柯山，亳州谯县(今安徽亳州市)人。北宋时期大臣、文学家，人称宛丘先生、张右史。代表作有《少年游》、《风流子》等。《少年游》写闺情离思，那娇羞少女的情态跃然纸上，让人羡煞爱煞，那份温情美妙真是有点“浓得化不开”。著有《柯山集》、《宛邱集》。词有《柯山诗余》。列为元佑党人，数遭贬谪，晚居陈州。
